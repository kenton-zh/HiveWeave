## 关键 Bug（按严重度）

### BUG-001 🔴 高 — `getThemeColors` race（navigate 后闪退"无项目"）
- **现象**：每次 `browser_navigate` 后 console 报 `[getThemeColors] TypeError: Cannot destructure property 'exportedColors' of 'undefined'`，UI 短暂退回"无项目"空状态约 1-2 秒
- **影响**：UI 操作不稳定，每次 click 之前要等闪回结束
- **位置**：`apps/web/src/components/office/OfficeView.tsx` 主题加载

### BUG-007 🔴 高 — Goals tab 切到后**完全空白**
- **现象**：点击 Goals 按钮后，右侧 panel 区域没有任何 GoalsPanel 内容
- **位置**：`apps/web/src/components/GoalsPanel.tsx`

### BUG-010 🔴 高 — **Executor 收件箱 → 触发执行 链路断裂**
- **现象**：CEO 把任务写进 A007 Engineer 收件箱（`unreadCount: 1`），但 60+ 秒后 A007 仍然：
  - `inbox` 消息 `read: false`（**没消费**）
  - `work-logs: []`（**没干活**）
  - chat history 没新消息（**没回复**）
- **影响**：**整个 Agent 执行链路不可用**——多 Agent 协作核心功能瘫痪
- **位置**：`apps/hiveweave-py/src/hiveweave/agents/supervisor.py` 或 `services/inbox.py` —— 缺监听 inbox → 触发 `agent.run()` 的循环

### BUG-009 🟠 中高 — **Mojibake / UTF-8 双重编码（DB 层）**
- **现象**：Engineer A007 的 role 在 3 个地方都是 `TypeScriptå·¥ç¨å¸`（应该是 "TypeScript工程师"）：
  1. `GET /api/org` 返回的 tree 节点
  2. HR→CEO 通信内容
  3. CEO→HR 的招聘需求原文
- **推测**：`str.encode('latin-1').decode('utf-8')` 或类似双重编/解码，发生在 `services/org.py` 写 DB 时
- **影响**：所有中文 role name 在 DB 里都是乱码，跨 agent 通信也带乱码
- **位置**：`apps/hiveweave-py/src/hiveweave/services/org.py`（推测）

### BUG-008 🟡 中 — Agent 工具调用参数 schema 不一致
- **现象**：CEO thinking 段暴露 —— 它先用 `path` 调 `read_file`，失败后改 `filePath`，还重复尝试了 7 次才把参数名搞对
- **推测**：tool executor 的参数名 schema 没有强制约束，LLM 自己摸索
- **影响**：每次 LLM 启动都要花 1-3 轮 tool-call 试错，**浪费 token** + 延迟
- **位置**：`apps/hiveweave-py/src/hiveweave/tools/executor.py` 工具 schema 定义

### BUG-002 🟢 **已证伪**（修订）
- **原假设**：暂停按钮无反应
- **实际**：正常，**是 async 同步延迟**（getPausedState 还没 resolve 时 UI 显示默认值）

### BUG-003 🟡 中 — ChatPanel 残留 `⚠️ 上次未收到回复`
- **现象**：进项目就看到红色警告
- **位置**：`apps/web/src/components/ChatPanel.tsx`

### BUG-005 🟡 中 — Polling 风暴
- **现象**：5 个组件各起一个 setInterval，navigate 时不取消
- **位置**：`OrgTree.tsx:725/763`、`ProjectTimeBadge.tsx:30`、`TodoBar.tsx:33`、`QuestionDialog.tsx:27`、`ChatPanel.tsx:506`

### BUG-006 🟡 中 — Navigate 取消 in-flight fetch 污染 console
- **现象**：每次 navigate 产生 5-7 个 `net::ERR_ABORTED` 进 console error stream
- **修复**：1 行 `if (err.name === 'AbortError') return null;`
- **位置**：`apps/web/src/api.ts:43` `fetchJSON`

### BUG-004 🟡 — Electron preload ENOENT（**不是 HiveWeave bug**，是 TRAE 安装包）
